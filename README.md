# WorldCup AI

WorldCup AI is an AI-powered FIFA World Cup 2026 platform that provides live scores, match statistics, player analytics, team insights, AI-generated summaries, predictions, and tournament intelligence.

## Features

### Live Match Center

* Live scores
* Match timelines
* Possession statistics
* Fouls
* Offsides
* Corners
* Yellow cards
* Red cards

### Team Analytics

* Team profiles
* Performance trends
* Tournament progression
* Historical comparisons

### Player Analytics

* Player statistics
* Match performance
* Rankings
* Player comparisons

### AI Features

* Match summaries
* Tactical analysis
* AI-powered chat assistant
* Match predictions
* Player insights

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* Supabase
* Vercel
* DSG World Cup API
* Free AI Models
